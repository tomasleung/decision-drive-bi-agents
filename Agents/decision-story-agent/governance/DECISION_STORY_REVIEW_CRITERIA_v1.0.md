# DECISION_STORY_REVIEW_CRITERIA_v1.0

## Purpose

Define the official review criteria used to evaluate:

```text
REPORT_STORY_MATRIX

REPORT_STORY
```

generated by the Decision Story Agent.

The criteria provide an objective review process before:

```text
Approval

Promotion

Template Updates

Downstream Agent Consumption
```

and establish a repeatable evaluation model independent of:

```text
LLM Vendor

Model Version

Agent Implementation
```

---

# Review Objectives

The review process exists to determine:

```text
Does the output satisfy the Gold Output Specification?

Does the output support business decision making?

Is the output complete?

Is the output traceable?

Is the output suitable for promotion?
```

---

# Review Outcomes

## PASS

Output satisfies:

```text
Gold Output Specification

Standards

Guidelines

Templates
```

No significant findings.

---

## PASS WITH REVISIONS

Output is usable but contains:

```text
Minor Gaps

Coverage Issues

Template Deviations
```

Corrections required before promotion.

---

## FAIL

Output contains major deficiencies in:

```text
Decision Design

Question Coverage

Signal Design

Traceability

Story Coverage
```

Regeneration required.

# REVIEW SUCCESS STATEMENT

A review succeeds when it objectively determines whether the artifact satisfies:

```text

Standards

Guidelines

Templates
```

while identifying:

```text
Gaps

Traceability Breaks

Coverage Issues

Template Improvement Opportunities
```
