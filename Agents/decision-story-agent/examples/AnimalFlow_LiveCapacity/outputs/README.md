# Outputs
## Decision Story Agent

---

# Purpose

This folder contains the official outputs generated by the Decision Story Agent.

These outputs transform:

```text
Business Requirements

↓

Decision Design

↓

Story Design
```

and provide the complete business design package required by downstream agents.

The outputs become the governing inputs for:

```text
Mockup Agent

TRD Agent

Semantic Design Agent

Report Build Agent
```

---

# Output Sequence

Outputs must always be generated in the following order:

```text
REPORT_STORY_MATRIX

↓

REPORT_STORY
```

Never skip an output.

Each output depends on information generated by the previous output.

# Output Governance

Outputs must:

```text
Remain Version Controlled

Remain Traceable

Remain Auditable

Remain Reviewable

Remain Business Focused
```

Outputs should not:

```text
Contain Technical Design

Contain Semantic Design

Contain DAX

Contain Data Modeling Decisions
```

Those responsibilities belong to later phases.
