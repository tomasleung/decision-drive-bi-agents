# SECTION 03 — BUSINESS LOGIC MODEL (high level)

### Capacity Availability

Capacity availability is determined by open spaces adjusted for utilization and confirmation status (Power App sign-off). Emergency closures override availability.

### Data Trust

Centres with missing assignment counts, stale updates, or low assignment accuracy must be flagged and excluded from automated routing until validated.
